# Заказное производство (Custom Production)

## Описание
Модуль для управления заказным производством. Отличается от серийного производства работой с поддонами вместо упаковок.

## Структура

### Основные компоненты:
- **CustomMasterPage.tsx** - главная страница заказного производства
- **OrdersTable** - таблица заказов (без изменений, как в серийном)
- **MachinesCards** - карточки станков (без изменений, как в серийном)
- **PalletsTable** - таблица поддонов (заменяет таблицу деталей)
- **PalletsSidebar** - боковая панель с деталями поддона
- **PartsModal** - модальное окно для работы с деталями (заглушка)

### Основные отличия от серийного производства:

1. **Таблица поддонов вместо деталей**:
   - Номер поддона
   - Материалы (детали в поддоне)
   - Адрес поддона
   - Статус поддона
   - Назначить станок
   - МЛ поддона (маршрутный лист)
   - Показать детали (стрелка)

2. **Создание поддонов**:
   - Кнопка "+ Создать поддон"
   - Выбор деталей из заказа
   - Детали можно выбирать только из текущего заказа

3. **Модальное окно с деталями**:
   - По клику на стрелку открывается боковая панель
   - Показывает список деталей в поддоне
   - Информация о поддоне (номер, адрес, статус)

4. **Данные**:
   - Пока используются дефолтные данные (моки)
   - API не подключено
   - Готово к интеграции с бэкендом

## Навигация

Переключение между серийным и заказным производством:
- В серийном производстве: кнопка в Sidebar (последняя иконка) → переход на `/custom-master`
- В заказном производстве: кнопка в Sidebar (первая иконка) → переход на `/master`

## Маршрут
`/custom-master` - доступен для ролей: admin, master
