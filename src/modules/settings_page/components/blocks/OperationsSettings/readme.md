# Структура компонента OperationsSettings

```
/src/modules/settings_page/components/blocks/OperationsSettings/
├── OperationsSettings.tsx - основной компонент
├── OperationsSettings.module.css - стили
└── components/
    ├── OperationsList.tsx - список технологических операций
    ├── OperationDetails.tsx - детали выбранной операции
    ├── OperationForm.tsx - форма добавления/редактирования операции
    ├── RelationsList.tsx - список связей между операциями
    ├── RelationForm.tsx - форма добавления/редактирования связей
    ├── MachineOperationsList.tsx - список связей с машинами
    ├── SegmentOperationsList.tsx - список связей с участками
    └── common/
        └── Notification.tsx - компонент уведомлений
```
