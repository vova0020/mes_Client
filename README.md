# FIT-MES Client

Этот проект использует React + TypeScript + Vite.

## Доступные команды

В директории проекта вы можете запустить:

### `npm run dev`

Запускает приложение в режиме разработки.\
Откройте [http://localhost:3000](http://localhost:3000) для просмотра в браузере.

Страница автоматически перезагружается при изменениях.\
Vite обеспечивает мгновенный Hot Module Replacement (HMR).

### `npm run build`

Собирает приложение для production в папку `dist`.\
Сборка оптимизирована для лучшей производительности.

### `npm run preview`

Локально предпросматривает production сборку.

## Переменные окружения

Переменные окружения должны начинаться с префикса `VITE_`:

- `VITE_API_URL` - URL API сервера
- `VITE_WS_URL` - URL WebSocket сервера
