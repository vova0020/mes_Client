# Изменения в модуле упаковки

## Дата: 2024

### Добавлена валидация по скомплектованным упаковкам

#### Проблема
Ранее оператор мог начать выполнение задания только когда все 100% упаковок были скомплектованы. Если было назначено 100 коробок, но скомплектовано только 92 - работу нельзя было начать.

#### Решение
Теперь оператор может выполнять задание частично на основе фактически скомплектованных упаковок.

#### Изменения на бекенде
Добавлены новые поля в ответ `/packing-assignments/by-machine/{machineId}`:
- `assembledQuantity` - количество скомплектованных упаковок
- `availableToComplete` - количество доступных для выполнения (min(remainingQuantity, assembledQuantity))

#### Изменения на фронтенде

**1. Обновлены интерфейсы:**
- `YpakTask` (ypakMachineApi.ts)
- `MachineTask` (machineMasterService.ts)

Добавлены поля:
```typescript
assembledQuantity: number;      // Скомплектовано
availableToComplete: number;    // Доступно для выполнения
```

**2. Обновлена таблица заданий (YpackMachinDetailsTable.tsx):**
- Кнопка "В работу" блокируется, если `availableToComplete === 0`
- Кнопки "Готово" и "Частично" блокируются, если `availableToComplete === 0`
- Добавлены подсказки при наведении на заблокированные кнопки
- При завершении задания используется `availableToComplete` вместо `remainingQuantity`

**3. Обновлено модальное окно "Частичная обработка":**
- Показывается "Осталось выполнить" (remainingQuantity)
- Показывается "Скомплектовано" (assembledQuantity)
- Показывается "Доступно для выполнения" (availableToComplete) - выделено синим
- Добавлено предупреждение, если не все упаковки скомплектованы
- Валидация: нельзя ввести больше, чем `availableToComplete`

#### Пример работы
**Сценарий:**
- Назначено: 100 коробок
- Выполнено: 8 коробок
- Осталось: 92 коробки
- Скомплектовано: 92 коробки
- **Доступно для выполнения: 92 коробки** ✅

**Сценарий 2:**
- Назначено: 100 коробок
- Выполнено: 8 коробок
- Осталось: 92 коробки
- Скомплектовано: 50 коробок
- **Доступно для выполнения: 50 коробок** ⚠️
- Показывается предупреждение: "Не все упаковки скомплектованы. Вы можете выполнить только 50 из 92 шт."
